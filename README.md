# Vera Health Chatbot

React Native chatbot with real-time SSE streaming for clinical questions.

## Demo

<video src="demo.MP4" width="300" controls></video>

## Tech Stack

- **React Native** + **Expo Router**
- **TypeScript**
- **SSE (Server-Sent Events)** for streaming
- **FlashList** for performance

## Quick Start

```bash
yarn install
yarn start
```

Press `i` for iOS, `a` for Android, or `w` for web.

## Features

- Real-time streaming responses
- Cancel mid-stream
- Markdown rendering
- Collapsible Q&A sections
